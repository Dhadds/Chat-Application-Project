# Chat-Application-Project

## Steps for the project

* `Step 1` : Set up .net core with entity framework for the backend and react/redux for the front end.

* `Step 2` : Set up a server using nginx

* `Step 3` : Set up mysql on that server and create a user table to store the username/password

* `Step 4` : install docker on your computer and get it running. 

* `Step 5` : Learn one way encryption with BCrypt

* `Step 6` : create a digital ocean droplet and get used to working on a remote server through ssh.

## Non Comprehensive List of Tools For The Project

* `Server Engine` : nginx https://www.nginx.com/

* `Container` : Docker

* `Container Orchestration` : Kubernetes (also known as k8s).

* `Automation` : CI/CD Continuous Integration and Continuous Development. GitLab.

* `Authentication` : JWTs for veriftying identities. 

* `Object-Relational Mapping` : maps database rows to objects.

* `Single Page Application` : SPA uses client side routing. 

* `RESTful API` : this is to be used in conjuction with SPA.

* `Socket.io` : this will help manage websocket connections.
